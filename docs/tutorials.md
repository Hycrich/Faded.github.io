# How to build your own simple Private Page

学习 CS 对我来说是一个很痛苦的过程，我对计算机的了解和掌握能力有限，最近又上到了 SYS II 和 ADS 这两个课程，写不出的 CPU 已经让我打算放弃学计科了，所以这一个月以来，我每天和自己说的最多的一句话就是“我要转专业”，我觉得也许在一个不那么底层的专业一切都会好的。所以我对现在愈发得讨厌，总是逃避、拖延，担心一事无成又毫无前进动力。不过，作业总是要交的，就像和它交手这么久总要有个结果，当然结果并不总是那么理想。

看到很多 CS 大佬整理的学习资源觉得很震撼，心血来潮就想着，为什么我不能有一个属于我的网站呢。不过当初搭建这个网站的时候我还是一个"green hand"，翻过无数篇帖子，纠缠着 deepseek 老师，真正做起来才发现，每个人的环境和遇到的问题时不一样的，debug 比真正搭建好框架更加复杂耗时。所以我决定将我的搭建过程写下来，一是帮助自己更系统理解这些命令都在干什么，二是希望对想搭建个人网站的伙伴们有帮助。
