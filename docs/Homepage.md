# Welcome to the Faded's

Faded 正在前往火星的路上。

宇宙很安静，孤独像影子般拉长，

飞船在黑色的海面漂荡，屏幕上闪烁着光。

燃料告急，短暂停靠，那是一颗名叫 GitHub 的小行星。

猫星人，悄无声息地出现了。

它们围在飞船旁边，没有询问目的，也不关来处，

只是递来无数能源晶体 —— 

`c` `python` `java` `html` `css` `javascript` `c++` `go` `rust` `swift` ...

Faded 开始整理那些匆忙扔上飞船的行李 ——

有没写完代码，

有抽象的概念，

有天马行空的想法，

也有无数瞬间，快乐、痛苦、感动...

引擎再次点亮，

Faded 要离他的火星更进一步了！

音乐、鲜花、烟火、彩带...

还有猫星人最衷心的祝福：

`git commit -m "meow"`

除了 —— `Faded.github.io`

这颗被留在宇宙中的碎片。

如果你恰好路过，

欢迎捡起它，

因为，当你读到这里时，

你宝贵人生中的一分钟已经属于我了。
