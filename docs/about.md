# About The Origin

本来打算把下面这段话写在 `welcome` 那个部分，不过刚进来就看些丧气话可不是很好，藏到 `about` 里面来吧。

进入大学的我就像河里的鱼游进了大海，在无限的世界里逐渐迷茫、挣扎、无所适从。当初因为迷茫选择成为一个 CS 人，现在却发现学习 CS 对我来说是一个痛苦的过程。我曾经以为我对计算机很感兴趣，但是底层的计算机逻辑却是那么的枯燥和晦涩，那些第一次发明底层架构、编程语言、操作系统的人简直是天才！我在计算机方面的了解和掌握能力和一起上课的大佬们比起来多么相形见绌，不巧最近又遇上 SYS II 和 ADS 这两个课程，每次上课听到这些底层逻辑我就有一种被催眠的感觉，完全听不进去；而写不出的 CPU 已经让我打算放弃学计科了，这一个月以来我每天都和自己说，“我要转专业”！我觉得也许在一个不那么偏底层的专业，一切都会好起来的。所以我现在总是逃避、拖延、摆烂，担心一事无成又毫无前进动力。

All right. 

所以这个网站的 origin 到底是什么？一个转变、一次尝试。为了写这个网站，当初那些没有系统掌握的知识我都需要重新去学习了解掌握。可当我再次翻开沉寂的课本，对着写下的笔记整理的时候，我发现他们或许并没有那么令人厌恶，我感到，对知识、生活的掌控感又一次在我心中上涨。从大二开始我已经很久没有这种掌控感了，每天似乎都在及时行乐和愧疚感的博弈中说服自己，但是似乎少了些意气，就好像人老了。现在的这种成就感是无以言表的，它更是一种幸福感，我成为了生活的主人，而不是随波逐流的过客。

最后，祝屏幕前的你、我，一路鼎沸！
